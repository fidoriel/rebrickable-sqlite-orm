# python LEGO Brick Database with ORM

This library loads the latest rebrickable database and creates a sqlite database.
All ORM Classes can be used to filter, query and aggregate through the data.
See `bootstrap.py` and `models.py` for downloading and usage.

Rebrickable states a `any purpose` license. Make sure to comply with licenses stated at https://rebrickable.com/downloads/.

Install with dev dependencies `uv pip install -e ".[dev]"`.
